Syntax

  • Comment ----> %%
  • Graph Direction
    • TB - top bottom
    • BT - bottom top
    • RL - right left
    • LR - left right
    • TD - same as TB
  • Node ----> node name1
    • graph LR id1
  • Node With Text ----> name node1[This is what will be inside the textbox]
    • graph LR id1[This is what will be inside the textbox]
  • Node shapes
    • Rounded corners ----> ()
      • graph LR id1(This is what will be inside the textbox)
    • Round node ----> (())
      • graph LR id1((This is what will be inside the textbox))
    • Asymetric Shape ----> >]
      • graph LR id1>This is what will be inside the textbox]
    • Rhombus ----> {}
      • graph LR id1{This is what will be inside the textbox}
  • Link Shapes
    • Arrow ----> -->
      • graph LR A-->B
    • Link Open ----> ---
      • graph LR A---B
    • Dotted Line ----> -.->
      • graph LR A-.->B
    • Thick Line ----> ==>
      • graph LR A==>B
    • Link with text Box ----> A---|This is the text|B;
      • graph LR A---|This is the text|B;
